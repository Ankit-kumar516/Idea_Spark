Project Title

Full Stack Web Application using React.js, Node.js, Express.js & MongoDB

1. Project Overview

A simple, clean and modern full-stack web application built for a hackathon. The app lets students share ideas and tasks, view others' submissions, comment, like and manage their own ideas. The goal is to demonstrate intermediate frontend skills (React.js) and basic backend skills (Node.js + Express.js) with MongoDB as the database.

2. Key Features

User registration & login (authentication)

Create, read, update and delete ideas (CRUD)

Like and comment on ideas

View feed / dashboard with ideas

Personal "My Ideas" section

Trending ideas & top contributors sidebar

Responsive UI with smooth animations and polished micro-interactions

Toasts, loaders and friendly empty states

3. Tech Stack

Frontend: React.js (functional components + hooks)

Styling: CSS / Tailwind / or CSS Modules (per preference)

Backend: Node.js + Express.js

Database: MongoDB (Atlas or local)

Authentication: JWT (JSON Web Tokens) or session-based (simple for hackathon)

Optional: Mongoose for MongoDB models

4. UI / UX Summary

Clean, minimal and modern look

Spacious layout, rounded corners, soft shadows

Smooth hover and click animations

Responsive across laptop, tablet and mobile

Fonts: Poppins/Montserrat for headings, Inter/Roboto for body

Important screens: Landing, Dashboard (Feed), Add/Edit Idea, Login/Signup, Profile
